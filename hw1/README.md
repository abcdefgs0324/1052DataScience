## HW1
To get the max value from given data.

<br>

### Usage
```
$ Rscript hw1_102703039.R -files [input-file] -out [result-file]
```

<br>

### Input Format
| persons | weight | height | gender |
| :-----: | :----: | :----: | :----: |
| person1 | 92.244 | 182.00 | F |
| person2 | 79.885 | 199.03 | F |
| ... | ... | ... | ... |

<br>

### Output Format
| set | weight | height |
| :-----: | :----: | :----: |
| [input-file] | [heaviest] | [highest] |
